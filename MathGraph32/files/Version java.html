Especificaciones de la versión java.


La versión Java posee todas las posibilidades ofrecidas para la versión Windows a excepción de la posibilidad de incrustar una figura en un documento guardando un vínculo OLE sobre este documento.

Es siempre posible incrustar una figura de MathGraph32 en un documento después de haberla copiado en el portapapeles  pero un doble clic sobre la figura no lanzará la versión Java de MathGraph32.

Se aconseja pues conservar una copia de la figura que se incrustará en el documento en el directorio que contiene al documento en caso de que desee modificarla más tarde.

Pero la versión Java posee también nuevos acciones:


  • MathGraph32 se convierte así multi plataforma.
  • La versión Java ha sido revisada y es más ergonómica.
  • Un nuevo ícono aparece: que permite modificar todo objeto gráfico haciendo aparecer el menú correspondiente.
    Es inútil entonces, en la versión Java,  decir cuál es el tipo de objeto que se desea modificar. Un clic sobre ese objeto con esta herramienta basta.
  • MathGraph32 Java permite poner un cálculo cualquiera en todo campo donde se espere un valor numérico. Por ejemplo se puede crear una circunferencia de radio 2*OA donde OA es una longitud que anteriormente se midió. Con la versión Windows habría sido necesario crear en primer lugar un cálculo nominado por ejemplo a y que contenía como fórmula 2*OA luego crear una circunferencia de radio a.
  • MathGraph32 Java permite abrir todas las figuras creadas con la versión Windows.
  • Los caracteres griegos y especiales se administran ahora de manera nativa: cuando ingrese texto en un editor donde algunos caracteres especiales están permitidos una caja de diálogo se abre automáticamente para permitirle escribir  los caracteres especiales.
  • MathGraph32 Java es personalizable: Se pueden autorizar sólo algunas herramientas. El programa informático puede así adaptarse al deseo del  profesor.
