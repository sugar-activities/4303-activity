Los íconos de la barra superior.

Para  crear una nueva figura provista de una longitud unidad.

Para abrir une figura a partir de un archivo.

Para guardar la figura en curso de edición.

Para anular la última acción sobre  la figura. La barra de información les indica cual era su naturaleza.

Para repetir la última acción anulada  sobre la figura.

Para zoom + sobre la figura.

Para zoom – sobre la figura.

Para revisualizar toda la figura recalculando las salidas aleatorias de la función rand() si la hubiera.

Para trasladar toda la figura deslizando el mouse.

Para capturar un objeto: punto libre, punto ligado, marca de ángulo, visualización de valor, comentario, título de macro.

Para nominar un punto o una recta. Si el nombre de un punto o una recta está próximo al puntero del mouse   podrá hacerlo deslizar.

Para cambiar el color o el trazado de un objeto (estilo de trazo, de punto, de flecha, de relleno).

Para modificar un objeto gráfico cuando se necesita una caja de diálogo para ser cambiado. Por ejemplo, cliquear sobre  un punto imagen de otro por una rotación hará aparecer una caja de diálogo permitiendo cambiar el ángulo de la rotación. Cliquear simplemente sobre  el objeto.

Para cambiar un objeto numérico.

Para destruir un objeto gráfico así como los que de él dependen.

Para destruir un objeto numérico así como los que de él dependen.

Para ocultar un objeto gráfico (el objeto existe pero no está visible).

Para hacer visible un objeto que ha sido ocultado. Los objetos ocultos aparecen dibujados en puntillado en su mayoría.

Para ver el historial de la figura. (atajo tecla F5).

Para ejecutar una macro efectuando un clic sobre su título.
